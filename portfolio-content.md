## PROJECTS

### Project 1 — Sandezi OA & Asset Management System

**Category:** Internal Tool / Enterprise Application
**Type:** Full-Stack Web Application
**Status:** Production deployed
**GitHub:** https://github.com/Chan030318/sandezi-oa

**Short description (60 words):**
Full-stack internal OA system for a live-streaming e-commerce company. Covers HR management, shift scheduling, leave approval, device asset lifecycle (borrow → maintenance → scrap), and 5-type CSV reporting. 29 PHP files · 11 DB tables · 3-role RBAC · defence-in-depth security.

**What it solves:**
The company managed 30+ staff and 50+ production devices via WhatsApp and Excel. No audit trail, no access control, constant scheduling conflicts, no visibility into device availability. This system replaced all of that with a single web platform any staff member can use without training.

**Tech stack badges:** PHP · MySQL · PDO · HTML/CSS · JavaScript · Apache · Git

**Key numbers:**
- 29 PHP files
- ~4,200 lines of code
- 11 database tables
- 3 user roles · 16 permission boundaries
- 29 CSRF check points
- 246 XSS prevention call sites
- 90 PDO prepared statements
- 5 CSV report types
- 22 git commits
