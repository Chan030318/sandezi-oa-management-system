# 三德子 OA 系统 - 第一阶段加强版

## 功能
- 登录系统
- Admin / Manager / Employee 权限
- Dashboard 数据看板
- 员工管理 + 搜索筛选
- 部门管理
- 班次管理
- 排班管理 + 基础冲突检测
- 我的排班
- 公告中心
- UI 美化

## 部署
1. 上传整个文件夹到服务器
2. 创建数据库 `sandezi_oa_plus`
3. 导入 `schema.sql`
4. 修改 `config.php` 的数据库账号密码
5. 打开 `login.php`
