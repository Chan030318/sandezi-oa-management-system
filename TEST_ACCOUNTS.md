## 四、测试前准备步骤

1. 执行 `deploy_production.sql` 初始化数据库（已含 Admin 账号和基础种子数据）
2. 用 Admin 登录，进入【员工管理】录入测试员工档案
3. 进入【用户管理】新增 Manager 和 Employee 账号，并绑定对应员工
4. 进入【场地管理】录入 2～3 个场地（如：A 直播间、B 拍摄间）
5. 进入【设备管理】录入 2～3 台设备（如：摄像机、麦克风）
6. 分角色使用对应账号登录进行功能测试
